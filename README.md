# Projeto de Site de Vendas de Automóveis

### Desenvolvido por Welington & Kauã

---

## Sobre o Projeto

Este projeto consiste no desenvolvimento de um site voltado para **exibição e comercialização de automóveis**.

A aplicação tem como objetivo apresentar veículos disponíveis para venda, permitindo que usuários visualizem informações detalhadas de cada carro de forma simples e intuitiva.

---

## Funcionalidades 

- Listagem de veículos disponíveis
- Visualização de detalhes dos automóveis:
  - Nome
  - Marca
  - Modelo
  - Ano
  - Estado de conservação
  - Preço
  - Imagens
- Acesso livre para usuários (sem necessidade de login)
- Painel administrativo (Admin):
  - Edição de informações dos veículos
  - Atualização de imagens
  - Alteração de valores e descrições

---

## Tipos de Usuário

### Usuário comum
- Acessa o site livremente
- Visualiza os veículos disponíveis
- Pode entrar em contato em caso de interesse

### Administrador (Admin)
- Responsável pela gestão do conteúdo do site
- Pode editar, atualizar e manter as informações dos veículos

---

## Contato

Caso o usuário tenha interesse em algum veículo, poderá entrar em contato diretamente com o administrador responsável pela venda.

---

## Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

- Prática de desenvolvimento web
- Estruturação de dados para exibição dinâmica
- Simulação de um ambiente real de vendas online
- Construção de portfólio profissional
